# 概要
前処理置換対象の原子要素をHTMLタグに置換する。

# 前処理対象原子一覧
| タイミング      | 要素     | 効果             | 対象文字、状況           |
| --------------- | -------- | ---------------- | -----------------------  |
| 前処理          | 原子     | 後ろ空白         | ！や？の後に空白(が無い) | 
| 前処理          | 原子     | 自動縦中横       | ！や？、英数の重なり     | 
| 前処理          | 原子     | 改行             | 改行コード               |
| 前処理          | 原子     | 空行             | 行頭改行コード           |
| 前処理          | 原子     | 全角空白         | 行頭以外の全角空白       | 
| 前処理          | 原子     | 半角空白         | 半角空白                 |
